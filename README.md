Hospital Finder

Hospital Finder is a React web application that allows users to locate nearby hospitals using Google Maps API. The app includes Google authentication, real-time location tracking, and an interactive map to display hospitals.

Features

 Google Maps Integration – Displays nearby hospitals based on user location.

 Google Authentication – Users can sign in with Google to access the app.

 Real-time Location Tracking – Automatically fetches user location.

 Nearby Hospitals – Lists hospitals within a 5km radius.

 User-friendly UI – Designed with Tailwind CSS for a modern look.
